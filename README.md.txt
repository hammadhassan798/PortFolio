## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/hammadhassan798/YourRepo.git
```

Install frontend

```bash
cd PortFolio
npm install
npm run dev
```

Install backend

```bash
cd backend
npm install
npm start
```
